## Задание 2
Данные (и Шум) - XOR

Входные признаки - X1, X2, X1*X2

Скрытые слои (Нейроны) - 0 слоев	

Активация - Linear

Основная цель (что наблюдать) -	Как перемножение признаков решает XOR
<img width="1584" height="856" alt="image" src="https://github.com/user-attachments/assets/23b4e22e-bd88-4721-83a3-fb4f3f7b286b" />

Количество затраченных эпох - 310

Test loss 0.004

Training loss 0.001

Вывод: Перемножение признаков (X1X2) решило задачу XOR, так как создало новый признак, который делает данные линейно разделимыми. Без скрытых слоев и с линейной активацией сеть по сути выполняла логистическую регрессию, но благодаря добавленному признаку X1X2 проблема из линейно неразделимой превратилась в линейно разделимую в трёхмерном пространстве признаков, что позволило сети достичь минимальных потерь.

## Задание 3
Данные (и Шум) - Спираль (Шум 20%)

Входные признаки - X1, X2

Скрытые слои (Нейроны) - 	3 слоя (6, 6, 6)	

Активация - Tanh

Дополнительные настройки - Regularization: L1, Regularization rate: 0.01

Основная цель (что наблюдать) -	Отключение нейронов регуляризацией L1
<img width="1561" height="857" alt="image" src="https://github.com/user-attachments/assets/79000545-f312-402b-bea6-efc5e05b7215" />

Количество затраченных эпох - 286

Test loss 0.501

Training loss 0.450

Вывод: L1-регуляризация (rate: 0.01) привела к тому, что многие связи между нейронами стали очень тонкими или исчезли, что означает обнуление их весов и фактическое отключение части нейронов. Это видно на визуализации — большинство линий почти невидимы, активны лишь несколько связей. Однако сеть всё ещё не смогла хорошо разделить спираль (test loss 0.501), так как L1-регуляризация упростила модель, отключив лишние нейроны, но задача спирали остаётся сложной даже с тремя скрытыми слоями
